# Hand-drawn presentation assets

Drop the CSS into the deck:

```html
<link rel="stylesheet" href="../Assets/handdrawn/handdrawn-assets.css">
```

Example use:

```html
<div class="handdrawn-mark handdrawn-circle" style="left:120px;top:80px;width:260px;height:150px"></div>
<div class="asset-postit asset-postit-coral" style="width:360px;height:304px">MarTech</div>
```

The hand-drawn SVGs use the talk palette: warm `#c84a2e`, cool `#355377`, ink `#1a1814`, and paper `#f3ede1`.

The post-it assets are transparent PNGs in `../postit/`.
